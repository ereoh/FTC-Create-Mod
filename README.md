# FTC Create Mod

## Installation

1. Clone this repo
  ```bash
  git clone git@github.com:ereoh/FTC-Create-Mod.git
  ```

2. Run `neoforge-xx.x.xxx-installer.jar` and install in your default Minecraft path (e.g., `C:\Users\Ere\AppData\Roaming\.minecraft`)
3. Create a Minecraft installation to use this repo
   1. In the Minecraft launcher top bar, go to the "Installations" tab (next to "Play")
   2. Press "new installation"
   3. Name it something like "FTC Minecraft"
   4. Set version to release-neoforge-xxx, with version matching the Neoforge currently in the repo
   5. Set game directory to wherever you cloned this repo; like `C:\Games\Minecraft\FTC-Create-Mod`
4. Launch game!
